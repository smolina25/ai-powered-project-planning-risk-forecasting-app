# Website Build Guides

These guides explain how each website version in the repository could be built manually by a human team, step by step, in a way that aligns with the capstone workflow.

They are written for a two-person team so you can learn:

- how the work could be divided between teammates
- what the manual technical workflow would look like
- what to say during the presentation if asked how the websites were built

## Versions Covered

- `v1` showcase microsite
- `v2` platform prototype
- `v3` lovable-style multi-page marketing site
