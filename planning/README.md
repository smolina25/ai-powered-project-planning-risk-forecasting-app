# Planning Workspace

This folder contains product-management planning artifacts that support the capstone but are not part of the main application runtime.

## Contents

- `aipm-product-discovery-planning/` - original AIPM planning, scoping, roadmap, and task breakdown materials

## Why this exists

These files were moved under `planning/` to keep the repository root focused on:

- application code
- tests
- deliverable docs
- presentation assets
- website variants

Use this folder when you want to review the PM process history behind the capstone.
